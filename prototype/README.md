# Story Driven Analytical System - UI Prototype

Прототип пользовательского интерфейса для Story Driven аналитико-информационной системы.

## Описание

Этот прототип демонстрирует основные UI компоненты системы, которая формирует сюжеты (stories) на основе:
- **Графа постов/новостей** (контентный слой)
- **Графа акторов** (люди, компании, страны, организации)

## Структура проекта

```
prototype/
├── index.html          # Страница сюжетов (Stories)
├── actors.html         # Страница акторов (Actors)
├── posts.html          # Страница постов (Posts)
├── data/               # Мокап данные в JSON формате
│   ├── stories.json    # Сюжеты и их метрики
│   ├── actors.json     # Акторы и их характеристики
│   ├── posts.json      # Посты/новости
│   └── relationships.json  # Связи между акторами
├── css/
│   └── styles.css      # Стили интерфейса
└── js/
    ├── app.js          # Общие утилиты и загрузка данных
    ├── stories.js      # Функционал страницы Stories
    ├── actors.js       # Функционал страницы Actors
    └── posts.js        # Функционал страницы Posts
```

## Функциональность

### Stories (Сюжеты)
- Список сюжетов с метриками (relevance, cohesion, freshness, size)
- Сортировка по различным критериям
- Детальный просмотр сюжета с:
  - Timeline (хронология фактов и мнений)
  - Список связанных акторов
  - Список постов в сюжете
  - Разбивка similarity metrics
- Поиск по сюжетам

### Actors (Акторы)
- Сетка карточек акторов
- Фильтрация по типу (person, company, organization, country)
- Фильтрация по тренду (up, down, stable)
- Сортировка по mentions, имени, дате обновления
- Детальный просмотр актора с вкладками:
  - Overview: общая информация
  - Mentions: упоминания в постах
  - Stories: связанные сюжеты
  - Relationships: связи с другими акторами
  - Aliases: известные алиасы
- Поиск по акторам и алиасам

### Posts (Посты)
- Список постов
- Фильтрация по sentiment (positive, neutral, negative)
- Сортировка по дате
- Связь с акторами
- Поиск по постам

## Запуск прототипа

Откройте `index.html` в браузере. Для корректной работы загрузки JSON данных рекомендуется использовать локальный веб-сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server

# Затем откройте в браузере
http://localhost:8000
```

## Мокап данные

### Stories (3 сюжета)
1. **GPT-5 Launch and Industry Response** - запуск GPT-5 и реакция индустрии
2. **AI Regulation and Safety Concerns** - регулирование и безопасность AI
3. **xAI Funding and Competitive Positioning** - финансирование xAI

### Actors (8 акторов)
- OpenAI, Sam Altman, Microsoft, European Union
- Google DeepMind, Elon Musk, xAI, US Congress

### Posts (10 новостей)
Новости связанные с AI индустрией, регулированием и конкуренцией

### Relationships (8 связей)
Различные типы связей между акторами:
- member_of, ally_of, competitor_of, part_of, owns, regulates

## Основные UI принципы

1. **Темная тема** - современный дизайн с dark mode
2. **Двухуровневая навигация** - Stories ↔ Actors взаимосвязаны
3. **Метрики и визуализация** - все ключевые показатели видны сразу
4. **Модальные окна** - детальный просмотр без перехода на новую страницу
5. **Фильтрация и поиск** - быстрый доступ к нужной информации
6. **Responsive дизайн** - адаптация под разные экраны

## Технологии

- Чистый HTML/CSS/JavaScript (без фреймворков)
- CSS Grid & Flexbox для layout
- Fetch API для загрузки данных
- Modern ES6+ JavaScript
